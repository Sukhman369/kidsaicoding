# Kids AI Coding Academy

A modern EdTech platform for kids to learn AI and coding. Built with CodeIgniter 4 and a clean, component-based frontend.

## Project Structure

- `app/`: Application core (Controllers, Models, Views, Config)
- `docs/`: Project documentation (PRD, TRD, DDD, Requirements)
- `public/`: Publicly accessible assets (CSS, JS, Images)
- `system/`: CodeIgniter 4 core framework
- `writable/`: Temporary files, logs, and cache
- `tests/`: Automated test suites

## Getting Started

1. Copy `.env` and configure your environment.
2. Run `composer install`.
3. Use `php spark serve` to start the development server.

## Knowledge Base

This project uses **Graphify** for indexing and knowledge management. Output is available in `graphify-out/`.
