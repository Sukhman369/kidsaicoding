# Kids Coding Academy Platform

## Frontend Design Specification (Version 1.0)

### Project Overview

Design a modern coding education platform for children aged 7-18 in India.

The platform provides:

* Live coding classes through Zoom
* Recorded courses through external LMS
* Assignments
* Progress tracking
* Certificates
* Parent monitoring dashboard
* Teacher management portal
* Student learning dashboard

Target Audience:

* Students (Age 7-18)
* Parents
* Teachers

Design Goals:

* Trustworthy for parents
* Exciting for children
* Professional for teachers
* Clean and simple navigation
* Mobile-first design
* Modern EdTech appearance

Avoid:

* Dark hacker themes
* Complex dashboards
* Excessive animations
* Overwhelming interfaces

Use:

* Bright but professional colors
* Rounded cards
* Friendly illustrations
* Achievement-focused visuals
